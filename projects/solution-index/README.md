# Smart Cyber Defence Team — Atomic Solution Index

This folder indexes isolated solution workspaces derived from the B9Tech Africa solution portfolio.

The root repository README remains the team landing page. Each solution must live in its own atomic project folder under `projects/`.

## Atomic Solution Rule

Each solution should be independently understandable, runnable, testable, and expandable.

A solution folder may contain:

```text
README.md
requirements.txt
src/
sample_data/
docs/
tests/
```
